20   * DATA_TYPE short =&gt; SQL type from java.sql.Types
21   * TYPE_NAME String =&gt; Data source dependent type name
22   * COLUMN_SIZE int =&gt; column size. For char or date types this is the maximum number of characters, for numeric or decimal types this is precision.
23   * BUFFER_LENGTH is not used.
24   * DECIMAL_DIGITS int =&gt; the number of fractional digits
25   * NUM_PREC_RADIX int =&gt; Radix (typically either 10 or 2)
26   * NULLABLE int =&gt; is NULL allowed?
27   * columnNoNulls - might not allow NULL values
28   * columnNullable - definitely allows NULL values
29   * columnNullableUnknown - nullability unknown
30   * REMARKS String =&gt; comment describing column (may be null)
31   * COLUMN_DEF String =&gt; default value (may be null)
32   * SQL_DATA_TYPE int =&gt; unused
33   * SQL_DATETIME_SUB int =&gt; unused
34   * CHAR_OCTET_LENGTH int =&gt; for char types the maximum number of bytes in the column
35   * ORDINAL_POSITION int =&gt; index of column in table (starting at 1)
36   * IS_NULLABLE String =&gt; &quot;NO&quot; means column definitely does not allow NULL values; &quot;YES&quot; means the column might allow NULL values. An empty string means nobody knows.
